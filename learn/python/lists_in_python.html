Working with Lists in Python

A list is a collection of items or values in Python. Lists are used to store multiple items in a single variable. They are one of the most important data structures in Python.

Creating a List

A list in Python is used to store multiple items in one variable. You create a list by enclosing the items in square brackets [ ] and separating them with commas.

For example:

        
# Creating a list of fruits
fruits = ["Apple", "Banana", "Cherry", "Mango"]
print(fruits)

Explanation:

  • The fruits variable is a list containing 4 items: "Apple", "Banana", "Cherry", and "Mango".
  • You can add, remove, or change items in the list later.
  • Lists are very useful when you want to work with multiple values together.

When you run this code, it will display:

        
['Apple', 'Banana', 'Cherry', 'Mango']

Accessing List Elements

In Python, every item in a list has a unique position called an index. The index helps us access individual items. Here are some important points to understand about list indexing:

  • The first item in the list has an index of 0.
  • The second item has an index of 1, the third item has an index of 2, and so on.
  • You can also use negative indexing to access items from the end of the list. For example, -1 is the index of the last item, -2 is the second last, and so on.

Here's an example:

        
# List of fruits
fruits = ["Apple", "Banana", "Cherry", "Mango"]

# Accessing items using positive indexing
print(fruits[0])  # First item: Apple
print(fruits[1])  # Second item: Banana

# Accessing items using negative indexing
print(fruits[-1])  # Last item: Mango
print(fruits[-2])  # Second last item: Cherry

Explanation:

  • fruits[0] gives the first item, "Apple".
  • fruits[1] gives the second item, "Banana".
  • fruits[-1] starts counting from the end, so it gives "Mango".
  • fruits[-2] is the second last item, so it gives "Cherry".

When you run this code, the output will be:

        
Apple
Banana
Mango
Cherry

Things to keep in mind:

  • If you try to access an index that doesn’t exist, Python will throw an error. For example, fruits[4] will cause an IndexError since there is no 5th item in the list.
  • Indexing is very useful when you want to work with specific items in the list.

Modifying List Elements

In Python, you can change the value of a specific item in a list by using its index. Here's how it works:

Example:

        
# Original list of fruits
fruits = ["Apple", "Banana", "Cherry", "Mango"]

# Change "Banana" to "Orange"
fruits[1] = "Orange"

print(fruits)

Explanation:

  • The original list is ["Apple", "Banana", "Cherry", "Mango"].
  • fruits[1] refers to the second item, "Banana".
  • We change "Banana" to "Orange" by assigning a new value: fruits[1] = "Orange".
  • After this, the updated list becomes ["Apple", "Orange", "Cherry", "Mango"].

Output:

        
["Apple", "Orange", "Cherry", "Mango"]

Adding Items to a List

You can add new items to the end of a list using the append() method. This is useful when you want to expand your list.

Example:

        
# Original list of fruits
fruits = ["Apple", "Banana", "Cherry"]

# Add "Mango" to the list
fruits.append("Mango")

print(fruits)

Explanation:

  • The original list is ["Apple", "Banana", "Cherry"].
  • The append() method adds the new item, "Mango", to the end of the list.
  • After appending, the updated list becomes ["Apple", "Banana", "Cherry", "Mango"].

Output:

        
["Apple", "Banana", "Cherry", "Mango"]

Removing Items from a List

In Python, you can remove items from a list using different methods. The two most common methods are:

1. Using the remove() method

The remove() method is used to remove a specific item from the list by its value.

Example:

        
# List of fruits
fruits = ["Apple", "Banana", "Cherry", "Mango"]

# Remove "Banana" from the list
fruits.remove("Banana")

print(fruits)

Explanation:

  • The original list is ["Apple", "Banana", "Cherry", "Mango"].
  • fruits.remove("Banana") looks for "Banana" in the list and removes it.
  • After this, the list becomes ["Apple", "Cherry", "Mango"].

Output:

        
["Apple", "Cherry", "Mango"]

2. Using the pop() method

The pop() method removes an item by its index. If you don’t provide an index, it removes the last item by default.

Example:

        
# List of fruits
fruits = ["Apple", "Banana", "Cherry", "Mango"]

# Remove the item at index 2 (third item)
fruits.pop(2)

print(fruits)

Explanation:

  • The original list is ["Apple", "Banana", "Cherry", "Mango"].
  • fruits.pop(2) removes the item at index 2, which is "Cherry".
  • After this, the list becomes ["Apple", "Banana", "Mango"].
  • If you use fruits.pop() without an index, it will remove the last item, "Mango".

Output:

        
["Apple", "Banana", "Mango"]

3. Important Notes:

  • If you try to remove an item that is not in the list using remove(), Python will throw an error.
  • pop() is useful when you know the position (index) of the item to be removed.
  • Both methods change the original list directly (in-place modification).
