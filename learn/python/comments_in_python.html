Comments in Python

What are comments? Comments are special notes that you write in your Python code. These notes are only for humans to read. The computer ignores comments when running your code.

Why Do We Use Comments?

Here are some reasons why we use comments:

  • To explain the code: Comments help others (and your future self) understand what your code does.
  • To make code readable: Well-commented code is easier to read and understand.
  • To test or debug: You can use comments to temporarily disable parts of your code for testing.

Types of Comments in Python

Python has two types of comments:

  • Single-line Comments: Use the # symbol. Everything after the # is ignored by Python.
  • Multi-line Comments: Use triple quotes (""" or ''') for longer comments. Python ignores everything between these quotes.

How to Write Comments?

Here's how you can use # and """ for comments:

1. Single-line Comments

Single-line comments start with #. Python ignores anything written after the #.

# This is a single-line comment
print("Hello, world!")  # This comment explains the print statement

Explanation: The comment # This is a single-line comment is ignored by Python and doesn’t affect the program. Only the print("Hello, world!") command runs.

2. Multi-line Comments

Multi-line comments are written inside triple quotes (""" or '''). They are useful for adding longer explanations.

"""
This is a multi-line comment.
It can span multiple lines.
Python ignores everything inside these quotes.
"""
print("Learning Python is fun!")

Explanation: Everything inside """ or ''' is ignored by Python, so it doesn’t affect the program.

Practice Example

Let's use both single-line and multi-line comments in a program:

# This program calculates the area of a rectangle

# Define the length and width
length = 5  # Length of the rectangle
width = 3   # Width of the rectangle

"""
Now we calculate the area.
The formula for area is length multiplied by width.
"""
area = length * width

# Print the result
print("The area of the rectangle is:", area)

What to do: Copy this code into Python and run it. Notice how comments help explain each step of the code.
