Understanding Variables in Python

A variable is like a container that stores information. Think of it as a box where you can keep something (like numbers or text) and give the box a name so you can use it later.

What is a Variable?

In Python, a variable is used to store data that you want to use or change later in your program. For example, if you want to remember someone’s name or age, you can store it in a variable.

How to Create a Variable

Creating a variable in Python is very simple. You just need to write a name for the variable, followed by the = symbol, and then the value you want to store. Here’s the format: 

variable_name = value

Here’s an example:

name = "John"
age = 25

In this example:

  • name: This is the variable name where we are storing the value "John" (a string).
  • age: This is another variable where we are storing the value 25 (a number).

Using Variables

Once you create a variable, you can use it in your program. For example:

name = "John"
age = 25

print(name)
print(age)

Output:

John
25

Rules for Naming Variables

There are a few simple rules you must follow when naming variables in Python:

  • Variable names can only have letters (a-z, A-Z), numbers (0-9), or underscores (_). 
      # Correct:
  valid_name = "Python"
  
  # Incorrect:
  invalid-name = "Error"  # Contains a hyphen (-)
  • Variable names cannot start with a number. 
      # Correct:
  name1 = "John"
  
  # Incorrect:
  1name = "Error"  # Starts with a number
  • Variable names cannot contain spaces. Use underscores instead. 
      # Correct:
  first_name = "Alice"
  
  # Incorrect:
  first name = "Error"  # Contains a space
  • Variable names cannot be Python keywords like print, if, or while. 
      # Correct:
  my_variable = "Allowed"
  
  # Incorrect:
  print = "Error"  # `print` is a Python keyword
  • Variable names are case-sensitive, so Name and name are different variables. 
      # These are two different variables:
  Name = "John"
  name = "Doe"
  
  print(Name)  # Output: John
  print(name)  # Output: Doe

Types of Data You Can Store in Variables

Variables can store different types of data. Here are some common types:

  • Text (String): Use quotes to store text, like "Hello" or 'Python'.
  • Numbers: You can store whole numbers (like 10) or decimal numbers (like 3.14).
  • True/False (Boolean): You can store either True or False.

Example:

name = "Alice"
age = 30
is_student = True

Why Use Variables?

Variables make your programs easier to write and read. Instead of repeating values, you can store them in variables and use them whenever you need. This makes your code clean, flexible, and easier to change.
